Jquery Lightbox++ Plugin

Beskrivning

Klicka på bilderna ovan för att se ett par exempel som använder sig av detta plugin.
Du kommer då att se den fullständiga storleken av bilden, samt en "caption" med bildens titel och beskrivning.

Läs vidare för att se installationsproccessen.

Om Lightbox++ Plugin

  • Innehållande filer

  • index.html - Ett html dokument som används för att visa detta exempel.
  • base.less - Ett bas css dokument som innehåller generella regler, skrivet i LESS.
  • style.less - Importerar base.less, här skrivs stylningen för denna sida och pluginet.
  • main.js - Pluginet är kodat i jQuery, här är koden som gör visningen av bilderna möjlig.
  • less.min.js - En kompilator fil för att kompilera LESS CSS till CSS.
  • jquery.js - Också självklart själva jQuery bibloteket !
  • Sex stycken olika jpeg bilder i olika färger. I storleken 200x200 pixlar.

Installation av Lightbox++

  • Provstart av pluginet

  • Kopiera mappen till din utvecklingsplats och index.html.
  • Om du själv vill styla pluginet och kanske använda det till ditt eget projekt så följ stegen nedanför.
  • Det räcker att konfigurera i 2 filer för att få resultat, style.less och main.js.

  • Stylning av Lightbox++

  • Öppna style.less i din text editor.
  • Lightboxen är uppbyggt utav divar som går att styla i style.less dokumentet.
    #overlay - Är den tonade bakrungsbilden som täcker hela bakgrunden.
    #lightbox - Detta element håller ihop bilden plus caption, caption innehåller i sin tur titel och beskrivning.
    .caption - Här går det att styla caption.
    .caption title - Stylning av titel.
    .caption desc - Stylning av beskrivning.
    .desc span - Är den lilla texten som talar om att man ska klicka för att stänga bilden.

  • Eget projekt

  • För att använda pluginet på din egen webbplats, gå in i katalogen js, välj main.js och jquery.js.
    Kopiera dessa två filer till ditt projekt och länka dem till din hemsida.
  • Som default inställning så letar pluginet efter img element i en div som heter main. Alla img element i div main kommer alltså att tillföras pluginet.
    För att ändra vilket div element som ska hanteras, öppna main.js. Gå till rad 6 och byt ut #main mot önskad #div.
  • I dina img taggar, var säker på att du anger önskad title inom attribut: data-title="TITEL" och önskad bildbeskrivning inom attribut: data-desc="BESKRIVNING"
  • Exempel på hur en img tag ska se ut.
  • <img src="img/6.jpg" data-title="Mörkblått" data-desc="En fyrkantig ruta." "alt="Test" width="100px" height="100px" />
